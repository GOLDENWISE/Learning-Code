Definisi

Sub-program yang dapat dipanggil pada bagian lain pada suatu program. javascript dibentuk oleh beberapa function. disebut juga sebagai prosedur untuk melakukan tugas atau melakukan perhitungan pada suatu nilai yang dibuat dari kumpulan statement

untuk menggunakannya, function harus dibuat untuk kemudian dipanggil nantinya.

Kategori

  • Built-in Function

    • Fungsi yang disediakan oleh javascript. contohnya ada pada syntax alert, prompt, confirm, dsb.

  • User-Defined Function

    • fungsi yang dibuat oleh kita sebagai programmer. cara penggunaanya adalah, diawali dengan function, diikuti oleh nama function(parameter). parameter berupa (nama variabel). parameter bersifat optional
    cth:
    function namaKu(nama){
    ...statement
    }
